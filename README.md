
### Principais recursos utilizados nos projetos:
Angular, SpringBoot, Node, Maven

### Principais linguagens utilizadas:
Java, type script, javascript, t-sql

### Principais banco de dados utilizados:
Sql server, oracle, postgres, mysql

### Projeto: api-vollmed
Projeto com autenticação JWT, aplicação de SOLID, Strategy Pattern, validações de negócio e versionamento de banco com Flyway

### 🧰 Tecnologias

- Java
- Spring Boot
- Spring Security
- JPA / Hibernate
- Flyway
- JWT
- SpringDoc (Swagger)
- H2 / Banco em memória (testes)

### 🧠 Conceitos aplicados
- Autenticação e autorização com JWT
- Controle de acesso por perfil (ROLE_ADMIN)
- Tratamento global de exceções
- Validações de regras de negócio
- Padrão Strategy
- Princípios SOLID
- Separação por camadas (Controller, Service, Repository)

### 🔐 Segurança
- Login com Spring Security
- Geração e validação de token JWT
- Filtro de autenticação (OncePerRequestFilter)
- Criptografia de senha com BCrypt

### 🧪 Testes
- Testes de integração
- Simulação de usuário autenticado
- Banco em memória para testes (H2)

### 🗄️ Banco de dados
- Versionamento com Flyway
- Migrations controladas por script
- Uso de JPA para persistência

### 📄 Documentação
- Documentação automática com SpringDoc (Swagger)
- Endpoints protegidos com JWT documentados

### ⭐ Diferenciais
- Implementação de regras de negócio complexas para agendamento e cancelamento
- Escolha automática de médico disponível
- Validações desacopladas usando Strategy
- Controle de exceções customizadas
